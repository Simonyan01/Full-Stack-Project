## _Features_

- _User registration and log in_
- _Authentication via JWT_
- _MySql Database_
- _Stripe subscription_
- _ORM Sequelize_

### Built by

- React.js
- Node.js
- Express.js
- CSS
- Visual Studio Code
- Postman
